ZoomBot is an automated utility designed to streamline Zoom meeting management and notification workflows.

## Features
* **Auto-Update Support:** Seamless updates for Windows, Mac and Linux users.
* **Cross-Platform:** Available for Windows, macOS, and Linux.
* **Secure:** Uses environment variables for sensitive API credentials.

## üöÄ Getting Started

### Installation
1. Download the latest version for your OS from the [Releases Page](https://github.com/LawrenceTanny/ZoomBot/releases).
2. Install and run the application once to generate the configuration folders.

### Configuration
You must provide your own credentials to run the bot. Place your `secrets.env` and `app_config.json` in the following directory:
* **Windows:** `%APPDATA%/ZoomBOt`
* **macOS:** `~/.config/ZoomBot`
